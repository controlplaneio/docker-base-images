# docker-base-images

Docker base images

# Build

```bash
make build
make build-all
make build-base
make build-base centos
make build-jre
```

# Test

```bash
make test
make test-all
make test-base
make test-base centos
make test-jre
```

# Push

```bash
make push
make push-all
make push-base
make push-base centos
make push-jre
```
